All source files are located in ECE474_Project1.
The main method is located in ECE474_Project1.cpp

To run have the text (example below is test.txt) file with the 
instructions and registers in the same directory as the 
ECE474_Project1.exe 

Command Prompt (In same directory)
ECE474_Project1.exe test.txt

Powershell (In same directory)
.\ECE474_Project1.exe test.txt

This project was built using Visual Studio Community 2019.
The latest microsoft visual c++ compiler was used that was
installed with Visual Studio Community 2019

It is unknown if this can compile on Visual Studio Community 2017
and function correctly.
